# 怎么不费力的做一个高情商的人？

> 不謙虛地講，一直以來無論男女老少，什麼類型什麼層次的人，他們都有一個比較一致的評價就是，這個女孩情商...

不謙虛地講，一直以來無論男女老少，什麼類型什麼層次的人，他們都有一個比較一致的評價就是，這個女孩情商很高，討人喜歡，今天我就用心理學的知識來跟大家分析一下，情商到底是什麼，怎麼練就，一個流傳比較廣的概念就是，一個人的成功是遵循二八法則的，20%的智商和80%的其他因素。

而這80%裡最重要的就是情商。所以情商到底是什麼，情商其實是情緒智力的簡稱，這個不是我說的，是情商之父，哈佛大學心理學博士，丹尼爾·戈爾曼，在一本叫情商的書裡面說到的，換句話說，情商是智力的一種，情商和智力是掛鉤的，智商是一個人的邏輯分析能力。

解決問題的能力，那麼情商就是裡面的一個小小的分支，是解決內心跟情緒有關，跟人際關係社會交往有關的智力，大部分人會覺得高情商的人應該很累吧，要壓抑自己要快樂別人，不僅要左右逢源，還要八面玲瓏，還要屈於尾翼背宮屈膝，但其實真正的高情商是一種敏銳的洞察力，是對人心和人性的洞察，是對對方情緒和需求的快速覺察。

它是一種非常強的綜合能力，丹尼爾·戈爾曼博士在情商這本書裡面，為如何從根本上提高自己的情緒能力，有非常清晰的四步法，瞭解自己的情緒，一個無法理解自己情緒的人，是必然無法理解他人的感受的，而那些真正高情商的人，他們無一例外，全部都是高共情能力的人，他們能很簡單的做到，想他人之所想。

你就是那句老話說的，將心比心，在這兒有一個非常簡單的訓練，叫做情緒顆粒度訓練，而這個顆粒度顧名思義，就是提高自己對自己情緒的精準感知能力，每次我們一旦產生某一種情緒，我們都要試著用非常精確的語言，把它表達出來，你可以自說自話，也可以寫日記，可以跟親朋好友講。

也可以跟心理諮詢師說，而這個訓練的辦法就是在每一次，你感到非常生氣或者難過的時候，問問自己給現在這個情緒，打分從1分到100分能夠打多少分，分之後再去問問自己，這次生氣是因為什麼原因而引發的，有沒有客觀上的誤會，有沒有主觀上認知的偏差，有沒有什麼自己可以需要調整的地方，其實這個東西就涉及到心理學上的一個概念，叫做原認知。

就是跳出框架之外去思考觀察自己，如果你能堅持這樣去觀察和研究自己的情緒，堅持以上的練習流程，很快你就會對自己的情緒感知能力有一個致的飛躍。第二步控制自己的情緒表達，這個就像我們去健身房，鍛煉自己的肌肉是一樣的，大腦的情緒控制也是需要訓練的，而這個訓練它包括不斷的思考和學習，這樣反向釋放，它需要一點情緒的控制和演技，是一個非常高階的辦法。

在你情緒即將釋放的時候，你嘗試著朝相反的方向去進行一個情緒的表達，高興的時候就是表現悲傷，生氣的時候就去展現溫和，緊張的時候去強裝淡定，而這種情緒的180度大轉彎的訓練辦法，它有利於你的情緒的認進，它會幫助你有一個更強大的情緒掌控能力，讓你能夠更靈活的對待不同的場景，同時能夠減輕別人對你的輕鬆動場和掌控的可能性，通過這個辦法，個體可以更好的掌握自己的情緒。

以免過度的表露有效的提高情傷。第三步和第四步就是識別和掌控別人的情緒，其實這個東西就變得很簡單了，因為高情傷不是簡單的說話做事讓人覺得舒服，而是擁有快速抵達對方心底的能力，前面兩步是關於自己，而後面兩步就是我們需要去觀察別人，去引導別人，去學會把自己的注意力放在別人身上，然後推及其人去準確的了解對方的真實想法和感受，還是那句話，高共情能力的人才能做到高情傷。

雙傷高的人春風得意，情傷高的人貴人相助。
