# 相貌平平，却让人痴迷？

> 長得不好看,但好像總是令對方痴迷，大家身邊有沒有那種人看起來笑貌平平，但好像在親密關係裡面特別吃得相...

長得不好看,但好像總是令對方痴迷，大家身邊有沒有那種人看起來笑貌平平，但好像在親密關係裡面特別吃得相混得開，他們不是那種長得特別漂亮,也不是那種長得特別帥的人，我總結了一下,他們好像擁有共同的特徵，這個特徵不是戀愛技巧,也不是套路，我一直都覺得小的套路是親密關係的良藥可以調節關係，而大的套路一定會讓你遠離真正的良緣，當你們真的去奉行這種戀愛的技巧，你去奉行那些網路的套路的時候,你就離正遠很遠很遠，而我一直覺得他們討喜的核心是說話好聽提供價值，包括但不限於情緒價值。

當對方想吃網紅餐廳的時候，走到門口發現人山人海還要等兩個小時的隊，你沒有不耐煩,你跟他說沒關係,我在這等，好吃的東西不怕的,我們打三把牌位,這個不就等到了嗎，又或者說是你們一起計劃去旅遊,對方的車票買錯了，明明是上海到浙江的,走到車站發現是浙江到上海的，這個時候你說沒關係,大家出來玩是很開心的事情，有這樣的意外只會讓你們的旅行增加更多的記憶點，對方東西買貴的時候,你跟他說沒關係,貴有貴的道理，偶爾咱們花一次這樣子的錢也沒有什麼大不了的事情，對方剛剛出入社會壓力很大,焦慮的時候你跟他講，人與人之間的發展階段是不一樣的,每個人都有自己的時機。

我相信你,你一定可以的,我陪你，什麼叫情緒價值,我這個講的情緒價值一定不是說這些假大空的話，去哄對方開心,而是真正擁有站在對方角度思考問題的同理心，這個不是了解,而是理解，我一直覺得了解和理解是兩個完全不同的意思，你了解對方的經歷不代表你真的能理解對方的不容易，而能真正提供到情緒價值的人是能夠站在對方的角度思考問題，真正的理解對方的不容易,真正的欣賞對方的閃光點，這樣你才能總是找到有的快的地方，他好像一直溫柔,永遠把我捧在手裡，我永遠能夠得到慰藉和支持，我好像在不怕外面的風雨,因為我知道你永遠陪著我。

我覺得這個才是真正的優秀健康的親密關係，才是現代婚戀的意義。除此之外,我覺得優秀的親密關係它一定不只是提供情緒價值，我不是讓你去做一個誇誇群的群組，而是你要擁有自己的專業技能,自己專業領域的知識，能夠給到他一些幫助，你要擁有自己人生智慧的美麗，能夠隨時在不同的視角、更高的維度，給到對方一些解讀或者是幫助，是永遠提醒對方的不足,鞭策對方進步，是在低谷的時候堅定的陪伴支持，也是在高峰的時候提醒對方一定要低調謙卑。

大家都說家和萬事興，我覺得和的前提一定是真正的理解對方,提供價值，做到真正的理解、尊重和支持，就讓大家忙碌一天回家之後，都擁有一個充滿愛和包容的家庭氛圍，因為這樣才是真的鬆弛、開心和幸福，正如那句話,貌美不如玲瓏心,閒妻扶我上青雲。好了,今天的分享就到這裡了，希望大家喜歡,再見。
