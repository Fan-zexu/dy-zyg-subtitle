# 越活越顺的人，背后的两点行事逻辑？

> 以前我觉得心狠是个贬义词，后来我是学了心理学，看了很多人物专辑，接触了很多现实生活中的大佬，我才知道...

以前我觉得心狠是个贬义词，后来我是学了心理学，看了很多人物专辑，接触了很多现实生活中的大佬，我才知道，原来他们风生水起，人生背后都离不开这两个字，注意啊，我说的是心狠，不是手辣，我不是让你伤害别人，而是让你对自己狠。

再注意啊，这个狠不是让你硬着头皮，逼着自己自律，而是像那些非常厉害的人一样，打心里面知道，自己是最重要的，那今天我会分享两点狠人心法，有需要的可以收藏学习。第一，做一个自私的人，矽谷的创投教父纳瓦尔，今年那期最精彩的演讲里面。

讲了一个非常重要的点就是，我们要学会自私，人生只有三万多天四千多周，不要去做不想做的事情，自私一点，把自己放在第一位，远离任何消耗的人和事儿，纳瓦尔本人是非常坚定的，信奉整体性自私的，他会毫不犹豫的把自己放在首位，就做个自私的人这句话，我觉得尤其对那些非常容易。

内耗的好人来说很适用，这很多时候你之所以会很累，你之所以没有力气做事情，搞事业搞身体，是因为你总是在为别人考虑，你总是在承接不属于你的情绪垃圾，你要知道心理咨询师，承接别人情绪也是要收费的，而你为什么要免费的，帮别人去做这些事情，还是那句话，人的精力是有限的。

用在内耗和替别人考虑上面，那么你的成长速度就会很慢很慢，而狠人之所以是狠人，是因为他们永远都把自己，当成一切事物的优先级。接着第二点，做一个勇敢的人，不要害怕失去，就恐惧这个情绪，是所有情绪里面最有毒的，就害怕会让我们卫手卫脚，左顾右攀小七八啦。

你们可以想象一下，就是职场上那些非常牛的人，情场上那些高手，有哪一个是身上沾了怂气的，甚至我可以说他们多多少少，都带着一点江湖气息，是废气，而真正的狠人，都是那种过去无悔，未来无畏的，他们不害怕任何一个人的离开，他们不害怕任何一件事情的落败。

听到这你们是不是觉得很夸张，是不是觉得道理我都懂。但是我就是做不到，那我觉得大家可以再停留一下，你可以想一下，为什么别人不怕，就你怕，是不是因为你没有领悟到，一个非常核心的点，就是这个世界上没有任何一个人，能陪我们走完一生，从头到尾我们都是一个人来。

一个人走，出现过的伴侣，父母，挚友，他都是一段一段的，缘起缘灭，就是这个世界的真相，我们无法对抗，以及我想再往深了一层说，很多人之所以怕。

其实是因为身体里面，留着弱势文化的血，就这个概念是我最近看，遥远的救世主这本神书里面学到的，中国人有一个很普遍的文化，叫做弱势文化，说的通俗一点就是，在家靠父母，出门靠朋友，一个靠子，就把咱们的弱的文化说尽了，而咱们今天聊的是啥。

是聊如何成为一个顺风顺水，扶摇直上，乘风破浪的人，如何成为一个狠人，而这个事情，咱们就得戒掉弱势文化，实体强势文化，我们能靠的从始至终只有自己，不要害怕失去，当你越是把这句话刻进你的大脑，你越会以常人无法理解的速度成长，你的能量也会越来越大。

而这个玩意儿，就是心理学的正向飞轮了，互相帮助，互相滋养，你越是不靠别人，你越强大，你越强大，你越是不靠别人，你越是不靠别人，你越不怕失去，你越不怕失去，你越强大。

好了我就不套话了，今天就分享到这儿，与君共勉。
