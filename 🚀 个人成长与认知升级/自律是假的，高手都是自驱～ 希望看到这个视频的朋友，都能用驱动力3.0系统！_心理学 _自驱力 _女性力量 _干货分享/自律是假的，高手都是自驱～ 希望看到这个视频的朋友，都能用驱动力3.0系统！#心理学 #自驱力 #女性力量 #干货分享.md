## 一、第一个故事的主人公是维基百科

在跟你们聊这个案例之前，我先问你们一个问题，假设现在时光倒退回到1995年，你现在旁边做了一个世界上特别牛的经济学家，投资者，你们手头上有两个项目，一个是成熟的商业帝国微软推出的电子百科全熟。另外一个是没有组织没有经费的维基百科，他们都在同一个赛道上竞争，你们觉得哪个项目会赢，故事回到1995年，那个时候的微软它已经声明确切了，是一个巨型的商业帝国了，而随着90年代科技的发展，在美国那个时候的个人电脑已经逐渐开始进入了很多普通老百姓的家里，电脑的普及让什么图片音频这些东西揉在一起变成了可能性。所以市场上对于在线电子百科全熟的需求，逐渐变得旺盛起来，而这个时候微软就推出了线上百科全熟项目，Inkarta，他高薪聘请了高级经理人，资深工程师，专家学者，内容编辑，一起编辑这个百科全熟，如果你是投资人经济学家，你大概率也会举双手赌微软，会在这个赛道胜出，但你们知道后来发生什么了吗？Inkarta虽然一开始还不错，有一些成绩。但是后来表现越来越差，直到2009年微软终止了这个项目，为什么万事俱备，有市场需求，有资金，有实力，微软还是没有做成。但是因为在2001年，维基百科横空出世，一开始它只是一个民间的公益网站，它没有正式的管理组织，没有钱，没有名声。但是故事的滑点就是，它用的是3.0模式，它开放编辑，人人都可以贡献内容，参与者没有报酬，只是本责对知识的热爱，以及追求那种，参与全球社区建设的归属感，就是这一系列的内在动机，让维基百科成为了一个，不断自我更新的庞大知识库，最终成为全球最大的，最受信任的在线百科全熟。

## 二、第二个故事就是刷蓝干

这个故事源于马克图恩的，汤姆索亚历险记，主人公汤姆接到了一个任务，就是把他的Poly A1家的，75平方米的柵蓝，当汤姆的朋友本，从蓝干路过的时候，汤姆并没有表现出，对刷蓝干的厌恶，反而他对本说，把颜料涂上渣蓝，并不是一件苦差事儿，而是一种特权，它是一个特别好玩的游戏，他就各种表现出，一副很享受刷蓝干的样子，让刷蓝干这件事情，看起来特别诱人，本就被汤姆的说法给吸引了，他就很好奇的试探性的问了一下，能不能让我来刷一下，汤姆就果断的拒绝了，但在他们两个人，来回的齐帆拉扯之后，Den终于用自己的苹果，作为交换条件，才获得了刷蓝干的机会，没多久，其他小伙伴们也加入进来了，他们都掉进了汤姆设置的圈套，全部都开始用自己的各种条件去置换，想要去体验这个刷蓝干的过程，并且还做得特别的有劲儿，这个故事就告诉我们，所谓的工作，就是一个人被迫要干的事情，至于玩嘛，就是一个人没有义务要干的事情，这个就是大名鼎鼎的，汤姆所要的效应，他揭示了一个道理，就是外在的奖励，会把本来好玩的游戏，变成枯燥的工作，相反，如果这个人觉得这个事儿很有趣，很独特，想把它做好，这种对专精的追求，反而会把枯燥的工作变成游戏。然后就是怎么更新游戏版本了，进入驱动力3.0时代了，记住三个词语，自主，专精，和目的，自主就是我们可以自主学习，自己喜欢的内容，用自己喜欢的方式，专精就是渴望把事情做得越来越好，不断地追求自我提升和个人成长，而当我们投入到自己热爱的事情当中，追求专精的时候，我们还有可能会体验到新流的状态，目的，也就是超越自身的渴望。
