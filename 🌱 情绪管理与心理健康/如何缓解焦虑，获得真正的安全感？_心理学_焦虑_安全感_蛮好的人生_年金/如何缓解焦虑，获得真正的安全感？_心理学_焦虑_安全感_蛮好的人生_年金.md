# 如何缓解焦虑，获得真正的安全感？

> 焦慮的本質其實是因為沒有退路和保障，我有一個特別好的朋友，31歲的女孩子，談了4年的戀愛，眼看就要結...

焦慮的本質其實是因為沒有退路和保障，我有一個特別好的朋友，31歲的女孩子，談了4年的戀愛，眼看就要結婚了，結果突然被對象斷崖式分手了，我原來以為她會很崩潰，我跑去安慰她，結果她非常平靜的跟我講，她說其實你不用安慰我，我有存款有副業有房子，就算她走了我也活得下去。

她還跟我說了一句我至今印象深刻，深刻到想把她改成我的朋友圈簽名的一句話，她說No one is coming，沒有人會來救你，真正的安全感是帳戶餘額和精神內核，你們知道嗎，她這個事和她這個人，讓我瞬間想起了心理學的一個概念，叫做內在安全感，什麼意思，就是說你是靠自己積累下來的，可控的資源。

會帶給你確定感，著名心理學家馬斯羅也說過，安全感是一個人最基礎的需求。但是你們有沒有想過，滿足安全感，從來不是一個人，也不是一段關係，而是你真真正正能夠掌控的東西，你的健康，技能，還有錢，就好像我這個朋友。

他以前年金的時候，掙了錢就特別喜歡買車買包，他現在就是，掙錢，攢錢，搞副業，買保險，他滿腦子想的都是提前退休，他還跟我推薦了他買的一個，支付寶上的一個叫，滿好的人生年金的產品，他說。

我要給未來退休的自己攢一個Plan B，我一開始其實挺好奇的，我在想是什麼產品讓他這麼有底氣，結果我自己去查才發現，這個是螞蟻寶聯合太平洋保險，推出的一款年金線，它的名字也是來自於我最近在看的，孫力的那個熱播劇叫滿好的人生，他講的其實就是一個概念，就是提前準備，才能從容過日子，我研究了一下。

我發現這個產品，它其實特別適合我們這種普通人。首先，收益都是白紙黑字寫信合同的，還是仙女姐姐劉亦菲代言的，螞蟻寶平台投保的，它又背靠國資委的太平洋保險城堡，說實話，這不比你對象的承諾來得心安嗎，如果你是30歲，存10萬，那麼到了5年末。

你就可以開始領錢，有7.5%的收益率，也就是說，你能拿到手的是7600塊的收益，而且從第6年年末開始，直到105歲，你每年都可以領取2481元，它的收益率其實是已經到了2.5%，已經遠超銀行了，關鍵是，它還是靈活存取，足期之後。

你隨時都可以自由的，無損的提出來，但有個事兒，我要提醒大家一下，保險利率，它這個東西，是跟著政策走的。所以它現在看起來是2.5%。

但是有可能年內就會降到2%，假如同樣是一次性投100萬，到85歲的話，這0.5%的差距，就賺出了將近30萬的差距。所以如果大家想買這個，蠻好的人生年金，真的可以越早越好，早點鎖住這個利率，然後穩穩的拿到收益，還是那句話，成年人的焦慮。

主要都是沒有保障，所導致的害怕，害怕未來不確定，害怕突然失業，收入和社保都中斷了，害怕沒有人愛，害怕沒有人陪，更怕的是30多歲的自己，還一事無成，其實我知道，就很多人雖然嘴上說著，我要躺平了。

但其實只有自己心裡知道，每天都在硬撐，心裡真的沒底。所以記住，真正的安全感，從來不是從某一個人，或者某一段關係裡來的，而是你為自己，提前準備的結果，就好比說給未來的自己的，一個底氣十足的，存錢分散筆。

最後，祝屏幕前的大家，吃喝不愁，早日退休，我們下期再見啦。
